2005

{% if LANGUAGE_CODE == "en" %}

The idea comes from Santa Felícia's teachers, when they notice that the childrens above 12 years old needed help.

{% else %}

A ideia nasce de professores do Santa Felícia quando estes perceberam que precisavam ajudar as crianças acima de 12 anos. Quando as crianças terminavam o projeto Recreação tinham poucas ou nenhuma opção saudável para ocupar seu tempo livre e ficavam mais perto de problemas sociais.

{% endif %}

2006

{% if LANGUAGE_CODE == "en" %}

--

{% else %}

Os professores realizavam atividades físico-desportivas como um poderoso instrumento de arregimentação desses jovens. Percebe-se que a necessidade de criar uma "marca" que representasse os interesses em comum das crianças e da comunidade.

{% endif %}

2006

{% if LANGUAGE_CODE == "en" %}

--

{% else %}

Escolheram um mascote para representar a entidade e suas atividades. Somos as ONG Formiga Verde, uma organização não-governamental fundada no dia 20 de março de 2006. E com atividades físico-desportivas a ONG inicia suas atividades.

{% endif %}

Hoje

{% if LANGUAGE_CODE == "en" %}

--

{% else %}

Atualmente a ONG Formiga Verde é composta por pessoas da comunidade, diversos voluntários e pelas comunidades atendidas. A ONG possui projetos próprios e pretente ampliar cada vez mais sua oferta de atividades.

{% endif %}
